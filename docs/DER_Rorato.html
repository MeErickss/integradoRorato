Relacionamentos

1 : N CATEGORIAS → PRODUTOS uma categoria possui vários produtos
1 : N CLIENTES → ORCAMENTOS um cliente solicita vários orçamentos
1 : N CLIENTES → MENSAGENS_CONTATO um cliente envia várias mensagens
N : N ORCAMENTOS ⇄ PRODUTOS resolvido pela tabela associativa ITENS_ORCAMENTO (um orçamento tem vários produtos; um produto aparece em vários orçamentos)

Legenda: PK chave primária · FK chave estrangeira · UK chave única.
